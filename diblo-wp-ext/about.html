Diblo Wordpress Extension

What can it do

Image

  • Reduce Chroma sampling to 4:2:0 and set image quality
  • Enable progressive format for images over 10 kilobytes
  • Change path to auto-generate images

CSS & Scripts

  • Handle browser default style during page loading
  • Organize and combine style and scripts

Backend

  • Block update of one or more plugins and themes
  • Re-order administrator menu and bar

Other

  • Handle prefetch
  • Disable Emojis
  • Redirect attachment without a parent
  • Analysis scripts (Google and Facebook)
  • Fix Yoast default Twitter image
  • Optimize 404 to 301 plugin MySQL table

Guide

Create a Must Use Plugin to add filters if a plugin has not already been created for that purpose.

Eks:

        <?php

        defined('ABSPATH') || die('No direct script access allowed!');

        add_filter( 'diblo_remove_generator_tags', '__return_true' );

        add_filter( 'diblo_chroma_sampling', function ( $array ) {
            [...]
            return $array;
        });

        add_filter( 'diblo_global_initialization_style', function ( $string ) {
            [...]
            return $string; 
        });

        add_filter( 'diblo_image_depth', function ( $int ) {
            [...]
            return $int; 
        });

Filters

diblo_remove_wlwmanifest bool false Remove the "wlwmanifest" link. wlwmanifest.xml is only needed for Windows Live Writer
diblo_remove_rsd_api bool false Remove the RSD. RDS is an API used for remote editing
diblo_remove_feeds bool false
diblo_remove_rest_api bool false
diblo_remove_shortlinks bool false Remove short link references. Short link reference in the document header indicates an abbreviated link to the page

diblo_remove_emojis bool false Disable WordPress emojis
